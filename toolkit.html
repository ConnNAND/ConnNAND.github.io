Godot Toolkit

Engine Used: Godot 4

Time Frame: Ongoing

Link: Github

About the Project

After completing several projects, I realized that there's a lot of things that I do basically the exact same way as soon as a start something new, so I made the decision to revisit some previous games, take some code, and integrate it all as new nodes into one Godot plugin that I can add to any project I start.

The initial spark for this was the audio systems I developed for Fear the Forgotten and Trash Lord. The plugin grew from there into tools to make things I find tedious easier.

Menus now have a single node to create and format them exactly how I like. Animation is slow and hard, so I developed a form of motion capture using Godot's animation system and a VR headset.

Dialog can be tricky, but plenty of games use it, so I build a dialog handler that supports audio playback and automatic scrolling as well. To go along with the dialog system I created a node for swapping animations or sprites based on phonemes detected in an audio waveform to make real time mouth movements without needing a pre-made animation. This also pairs with an eye tracking node that moves a pupil to look at a point of interest, both in 2D and in 3D.

The true cherry on top of all of this is the music creation tool I developed to put my knowledge gained from my Music Theory 2 course into action. It allows you to build a series of chords and easily manipulate them without the use of a piano roll, which I found helps you focus on the principles of music theory more so than the individual notes at play, then you can export the sequence to a midi file and drag & drop it into your DAW of choice.

Accomplishments & Lessons Learned

Creating this plugin salvaged some of the best parts of my previous projects or projects that had never been finished/released, allowing me to skip tediously creating the same things over and over again, I could revisit their flaws and improve upon them, as well as creating foundations for new projects and tools to use in the future to make life easier, and it keeps growing as I grow.
